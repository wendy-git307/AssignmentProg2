# prog2AdoptionCentre 

Welcome to the **Prog2 Adoption Centre** application! This JavaFX-based project is designed to simulate an animal adoption center, allowing managers and customers to interact with the system to manage and adopt animals.

## 🐾 About the Project

The **Prog2 Adoption Centre** application provides a platform for managing animals in an adoption center. It supports two types of users:
1. **Managers**: Can add or remove animals and view the list of users
2. **Customers**: Can view available animals, adopt them, and see their adoption details

---

## ✨ Features

### For Managers:
- View all animals in the adoption center
- Filter animals by type (Cats, Dogs, Rabbits)
- Add new animals to the adoption center
- Remove animals from the adoption center
- View the list of registered users

### For Customers:
- View available animals for adoption
- Adopt animals (with a limit of 2 per type)
- View details of adopted animals

### General:
- Error handling for invalid operations
- Dynamic updates to the animal list

## 🛠️ Technologies Used

- **Java**: Core programming language
- **JavaFX**: For building the graphical user interface
- **FXML**: For designing UI layouts
- **CSS**: For styling the application

---

### Key Files:
- **`Prog2AdoptionCentreApp.java`**: Entry point of the application
- **`controller/`**: Contains all the controllers for handling UI logic
- **`model/`**: Contains the business logic and data models
- **`view/`**: Contains FXML files for UI design and `style.css` for styling
- **`au.edu.uts.ap.javafx`**: Utility classes for loading views and managing controllers

## 🚀 How to Run

### Prerequisites:
- Java Development Kit (JDK) 8 or higher
- JavaFX SDK
- An IDE like **Visual Studio Code** or **IntelliJ IDEA**
